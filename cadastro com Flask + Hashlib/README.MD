# Cadastro com Flask + Hashlib

Projeto simples de cadastro de usuários usando Flask.

A ideia aqui foi treinar backend básico com Python, criação de rotas e também um pouco de segurança usando hash de senha.

## O que tem no projeto

- Cadastro de usuário
- Validação básica de dados
- Armazenamento de senha com hash (hashlib)
- Uso de Flask pra criar rotas e lidar com requisições

## Tecnologias

- Python
- Flask
- Hashlib
- HtML
- Css
